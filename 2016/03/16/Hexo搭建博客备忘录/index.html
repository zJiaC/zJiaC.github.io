三、npm相关

  • 起因：
    1. 因为国内有墙的原因，所以大部分时候安装某些包的时候并不是那么的容易，
    2. 常见的是利用vpn翻墙出去，修改dns和host，在或者修改npm的源。
    3. 而nrm就是用来切换源的。
  • 安装
    npm install nrm -g
  • 使用
    nrm ls //查看可以使用的源
    nrm test //测试源的连接时间
    nrm use “源的名字” //切换到指定的源
  • 使用
    gnvm 可以用于管理node版本
